# HMT Escrow Form
Purely based in javascript now, interacts, validates automatically from the blockchain.

Modify [config file](client/src/config.js) and set the following variables:
```
ethProviderUrl: '',
hmtokenAddr: ''
```

Install Dependencies:
```bash
cd ./client
yarn install
```


Run the react app:
```
yarn run start
```

go to localhost:3000

Production build:
```
yarn run build
```
